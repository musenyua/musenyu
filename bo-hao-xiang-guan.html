拨号相关

部署脚本

curl -o /opt/bcloud/scripts/bxc_deployment.sh https://lego.s3.cn-east-2.jdcloud-oss.com/scripts/bxc_deployment.sh && chmod +x /opt/bcloud/scripts/bxc_deployment.sh && echo "alias bxc_deployment='sh /opt/bcloud/scripts/bxc_deployment.sh'" >> /etc/profile && source /etc/profile

查看拨号是否完整

ip r |grep ppp|wc -l

重播

for i in $(ls /etc/sysconfig/network-scripts|grep ppp[0-9]);do ifup $i & done

移动内网查询是否nat1

pip install pystun
for i in `ip r|grep ppp|awk  -F' ' '{print $NF}'`;do pystun -i $i ;done
