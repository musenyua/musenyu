制作iperf测速服务器

拉取执行脚本

sudo wget -O /opt/yg/iperf.sh http://musenyu.cn:8881/iperf.sh && sh /opt/yg/iperf.sh

sudo wget -O /opt/yg/reiperf.sh http://musenyu.cn:8881/reiperf.sh && sh /opt/yg/reiperf.sh

修改定时处理任务

sudo vim /etc/crontab

加入

0 */1 * * * root /opt/yg/reiperf.sh >> /dev/null 2>&1

重启定时任务

sudo service crond reload
